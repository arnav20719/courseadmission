# CourseAdmission - College Admission Portal

## Project Overview
A full-stack web application for college admission inquiries, built with Next.js, Prisma, and Supabase.

## Live Demo
https://courseadmission.vercel.app

## Features
- Lead Capture System - Students fill name, phone, course interest
- College Directory - Browse 9+ colleges with details
- Course Listings - View courses offered at each college
- Search & Filters - Filter by state, stream, type
- Admin Dashboard - View and export leads to Excel
- Bihar Credit Card - Education loan scheme information
- Chatbot Integration - AI assistant for student queries

## Tech Stack
- Next.js 16
- JavaScript
- Prisma
- Supabase (PostgreSQL)
- Tailwind CSS
